## 🗄️ Banco de Dados:

### Tabelas (4 principais):
1. **empresas** - Dados das empresas
2. **pesquisas** - Pesquisas realizadas
3. **avaliacoes** - Avaliações por área
4. **indicacoes** - Indicações de clientes

### Relacionamentos:
- pesquisas.empresa_id → empresas.id
- avaliacoes.pesquisa_id → pesquisas.id
- indicacoes.pesquisa_id → pesquisas.id
